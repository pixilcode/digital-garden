**Link: [You can code only 4 hours per day](https://newsletter.techworld-with-milan.com/p/you-can-code-only-4-hours-per-day)**

A developer can only productively code for 4 hours each day. This is because of a cognitive ceiling that is easily reached. In addition, there are many distractions that occur throughout the day that take us out of the flow state. Therefore, it's important that we make efforts to create a space where we can enter a flow state.
