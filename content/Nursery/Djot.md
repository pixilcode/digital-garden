[Djot](https://djot.net/) is a markup syntax similar to Markdown. However, it improves upon Markdown by adding rules and features that make it less ambiguous and easier to parse.

[Syntax reference](https://htmlpreview.github.io/?https://github.com/jgm/djot/blob/master/doc/syntax.html)

**Features that I'm excited about:**
- math support
- footnotes
- generic spans/divs
- inline/block attributes (classes, ids, other html attributes)
- links to headings
